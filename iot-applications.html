Topic 1

Activity 1: IoT Apps

What is an IoT application?
 An application that interconnects devices so they can communicate with each other without any human interference.

What are the main features needed?

  • Sensors and Devices
  • User Interface
  • Security and privacy systems​
  • Communication – different kinds of networks​
  • Data Management (with Cloud Computing)
  • Applications – toolboxes, frameworks, software​

    • IoT Applications Examples:

  • Traffic Monitoring Systems
  • Wearables (smart watches)
  • Transportation (driverless cars)
  • Security Systems
  • Military drones
  • Smart buildings

    • Activity 2: Smart Building

    What does smart mean?

  • Capture and utilize every bit of data which you share or use in everyday life​
  • Sense, think, act – take in the data, process it and respond accordingly​
  • Able to integrate with other devices​

    • Main Features Needed:

  • Storage, collection of data​
  • Cloud Computing​
  • Sensor Technology
  • Wireless Communication
  • Battery/Power source
  • Processing
  • Centralized event processing: execute functions to monitor, navigate and optimize processes in real-time​

    • Smart Building Scenarios

  • Temperature control​
  • Automated Lighting​
  • Lighting Management
  • Security Systems
  • Monitor Power/Energy Usage

    • Activity 3: Control Interface

    What are the high-level features?

  • Hardware - Multiple IOT devices with varied capability levels
  • Processing - Raspberry pi, Intel Edison…
  • Connectivity - WIFI, Bluetooth modules
  • Data Management - Cloud Computing, Storage devices

    • Parts Needed:

  • Memory​
  • Sensors
  • Actuators
  • Housing for the device
  • Input/Output Ports​
  • Microprocessor

    • Who are the Users?

  • Large Corporations (industrial IoT applications)
  • People who are not much familiar with the technology (consumer IoT applications)
  • System administrators (authorization)
  • Management – auditing and finance
  • Device manufacturer (debugging and updates)
